Billentyűkombinációk

A leghatékonyabb parancsok az irodai munkavégzéshez.

Rendszer és Navigáció

  • Ctrl + Shift + T: Bezárt lap visszaállítása
  • Ctrl + Z: Visszavonás
  • Win + Nyilak: Ablak rendezése
  • Ctrl + Görgő: Nagyítás / Kicsinyítés
  • Ctrl + C / V / X: Másolás, Beillesztés, Kivágás
  • Win + D: Asztal megjelenítése
  • Win + L: Gép zárolása
  • Win + Shift + S: Képernyőmetszet
  • Alt + Tab: Váltás ablakok között

Böngésző Kezelés

  • Ctrl + T: Új lap
  • Ctrl + W: Lap bezárása
  • Ctrl + F: Keresés az oldalon
  • F5: Oldal frissítése
  • Ctrl + L: Címsor kijelölése

Szerkesztés

  • Ctrl + B / I / U: Félkövér, Dőlt, Aláhúzott
  • F2: Átnevezés
  • Win + .: Emoji választó
