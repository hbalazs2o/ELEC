Vájb Kód Bevezető

Üdvözöllek a Vájb Kód világában! Ez a modul végigvezet az Antigravity AI használatán és a modern webes fejlesztés alapjain.

1. Kezdés az Antigravity-vel

Az Antigravity egy forradalmi AI kódoló asszisztens, amely közvetlenül a fájljaid között dolgozik.

  • 01 Nyisd meg a böngésződben az Antigravity felületét.
  • 02 Csatold a munkakörnyezetedet (Workspace) az AI-hoz.
  • 03 Hitelesítés után egyszerűen csak írd le magyarul a feladatot.
  • 04 Az AI azonnal generálja a kódot, amit rögtön tesztelhetsz is.

2. A Webfejlesztés Alapjai

A) HTML (Szerkezet): Ez adja meg az oldal vázát.

<div class="card">
  <h1>Saját Jegyzet</h1>
  <textarea id="note"></textarea>
</div>

B) CSS (Stílus): Ettől lesz prémium a megjelenés.

.card { 
  background: white; 
  border-radius: 20px; 
  padding: 20px; 
  box-shadow: 0 10px 30px rgba(0,0,0,0.1); 
}

C) JS (Logika): Ez teszi interaktívvá az oldalt.

function save() { 
  localStorage.setItem('myNote', document.getElementById('note').value); 
}

3. Hosting a GitHub-on (Élesítés)

Hogy mások is láthassák a munkádat, a GitHub Pages a legegyszerűbb megoldás.

  1. Regisztrálj a github.com oldalon.
  2. Hozz létre egy új "Repository"-t.
  3. Töltsd fel a fájljaidat, ügyelve az index.html fájlra.
  4. A Settings -> Pages menüben állítsd be a forrást.

PRO TIPP: Használd az Antigravity-t a kódolás közben: "Húzz egy üveg-effektust a kártyákra!" - és ő megcsinálja helyetted.
